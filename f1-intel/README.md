# F1 Intelligence Platform (Vercel-First)

This is a serverless Next.js 14 build that implements:
- Telemetry Pit Board Dashboard
- AI Driver Coach (post-lap analysis)
- AI Race Summarizer
- Race Result Predictor

## Local dev

```bash
cd f1-intel
npm install
npm run dev
```

## Environment variables

Create `.env.local` in `f1-intel/`:

```
SUPABASE_URL=
SUPABASE_KEY=
OPENAI_API_KEY=
OPENAI_MODEL=gpt-4.1-mini
```

## Supabase tables

```sql
create table if not exists telemetry_cache (
  id bigint generated by default as identity primary key,
  session_id bigint not null,
  driver text not null,
  lap int not null,
  lap_time numeric,
  sector_1 numeric,
  sector_2 numeric,
  sector_3 numeric,
  created_at timestamptz default now(),
  unique (session_id, driver, lap)
);

create table if not exists race_summaries (
  id bigint generated by default as identity primary key,
  race_id text not null,
  summary text not null,
  created_at timestamptz default now()
);

create table if not exists predictions (
  id bigint generated by default as identity primary key,
  race_id text not null,
  driver text not null,
  predicted_position int not null,
  confidence numeric not null,
  created_at timestamptz default now()
);
```

## Routes

- `/pitwall`
- `/coach`
- `/summary`
- `/predictor`

## API

- `GET /api/telemetry?session=latest&driver=VER`
- `POST /api/coach`
- `POST /api/summary`
- `POST /api/predict`

## Vercel

Set the Vercel project root to `f1-intel`.
